Ten Brush Sizes Plugin (v1.0.1)

Provides shortcuts for setting the brush to my favorite sizes (hardcoded for now and subject to change).

Usage

Go to Settings => Configure Krita => Keyboard Shortcuts => Search "Set Brush Size" and set to whatever shortcuts you want. Personally I use keys 1 through 0.

There are also now two aditional shortcuts "Increase/Decrease Brush Size (Custom)" that use the set sizes.

To configure the sizes, go to Tools => Scripts => Ten Brush Sizes
